# Seasons of the Rustle Town

2D 像素风农场生存游戏（Godot 4.7）。

## 快速开始

1. 安装 [Godot 4.7](https://godotengine.org/)
2. 打开本目录（含 `project.godot`）
3. 按 **F5** 运行
