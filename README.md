# The Seasons of the Rustle Town

> 一款类星露谷风格的模拟经营游戏，融合放置、种植、伐木、钓鱼、采矿与地牢战斗。  
> 像素风格，精致光影，美国田园小镇氛围，碎片化叙事，不打断经营节奏。


##  核心玩法

- **种植**：主要经济来源，作物收益最高
- **伐木 / 钓鱼 / 采矿**：基础资源采集
- **战斗**：主要在矿洞与地牢中进行
- **探索**：矿洞内隐藏彩蛋、秘密区域与特殊敌人

##  战斗系统

- 近战武器 + 远程武器
- 战斗区域：矿洞、地牢
- 世界观内存在魔法元素

##  探索与彩蛋

- 矿洞内可炸开巨石，发现隐藏区域（如秘密实验室、巨型机械怪物）
- 天上可能掉落：陨石、废弃航天器
- 所有地名均为虚构

##  美术与风格

- **画面**：像素风格
- **光影与水**：精致处理，增强氛围
- **整体风格**：美国田园小镇（虚构设定）

##  叙事方式

- **碎片化叙事**
- 不追求宏大剧情
- 构思剧情的原则：不能影响模拟经营的核心体验

##  经济系统与正反馈循环

1. **种植** → 获取大量金钱
2. **金钱** → 购买初始道具
3. **道具** → 进入矿洞/地牢冒险
4. **冒险产出**：
   - 宝箱
   - 原材料
   - 稀有植物种子
5. **原材料** → 提炼 → 制造农业器械
6. **农业器械** → 提升种植效率

> 目标：形成微弱的正反馈循环，让玩家逐步成长，但不失控。

##  NPC 设定
##  三类角色可以有重合
##  NPC男女比例尽量1：1，无论是功能性角色还是剧情角色，否则会破坏日常感，NPC分成3个年龄段，主角的父辈，主角的同辈，儿童，每个人都有小支线

### 第一类：功能性角色（贸易与服务）

这些角色负责游戏的经济与服务功能，但拥有自己的生活和性格。


**超市老板** 
基础商品贸易，小镇消息最灵通的人，爱八卦，带点奸商属性

**宝石商人**
收购宝石，出售稀有饰品，神秘，偶尔透露矿洞深处的秘密 

**怪物商人** 
收购怪物掉落物，出售稀有材料，怪异但友善，曾是个冒险者

**铁匠**
武器打造与升级，有着铁匠标志性的牛角胡，沉默寡言，手艺精湛 

**木匠**
房屋升级、农场建筑 

**巴士司机** 
区域交通 

**船长**
水路交通、特殊区域解锁，脾气古怪，有个失散多年的弟弟

**博物馆馆长**
同星露谷物语的Guther

> 他们和剧情类角色之间可能有亲属关系

### 第二类：剧情推进角色

- **逃难者**（住在森林小屋）
  - 异国他乡口音，来历不明
  - 身上似乎藏着某种秘密

- **疯狂科学家**（拥有巨大的地下实验室）
  - 发现逃难者的特别之处
  - 委托玩家调查他
  - 奖励：黑科技小发明（特殊道具/装备）

- **两位联邦探员**
  - 原本便衣潜伏抓捕某人
  - 为了装酷天天穿黑风衣戴墨镜
  - 潜伏半年毫无进展，现已摆烂，天天泡吧

- **废弃航天器的幸存AI**
  - 从天上坠落的废弃航天器中残存的人工智能
  - 失去大部分记忆，只记得自己“曾经很重要”
  - 玩家修复它后（通过完成科学家的委托获得维修道具），会逐渐解锁隐藏剧情线索
  - 奖励：科技树解锁、自动化农场组件

- **地牢底层的神秘女巫**
  - 被封印在地牢最深处，声称自己是被冤枉的
  - 玩家可以选择：释放她 / 无视她 / 向探员举报她
  - 不同选择触发不同支线
  - 奖励：魔法配方、诅咒装备（高风险高回报）

- **游走的古董商（不定期出现）**
  - 行踪不定，只在特定天气或季节出现
  - 收购古董级物品，出售罕见种子和图纸
  - 他手里的物件似乎不属于这个时代
  - 暗示：他和科学家、航天器之间有某种联系

- **气象站的怪人**
  - 独自住在山顶气象站，研究异常的天气现象
  - 发现天上的陨石和航天器似乎不是偶然掉下来的
  - 需要玩家帮忙收集坠落物的碎片
  - 奖励：天气预测装置、特殊肥料配方

- **湖底的沉睡者**
  - 钓鱼时极小概率钓到的神秘信物
  - 触发后解锁湖底洞穴
  - 里面沉睡着某个超古代的存在，尚未完全苏醒

###  第三类角色：可攻略角色
年龄在16-25之间
